# NEON // TALES

A self-contained, local-first light novel publishing prototype. Open `index.html` in a browser to use it. The app has no build step or package install.

## Included

- Landing page with featured release, archive grid, and live writing stats
- Series detail screens with tags, status, release cadence, and chapter archive
- Immersive reader with type size, font, and dark/light/sepia controls
- Author Console to create series and publish chapters
- Browser-local storage: additions stay available after refresh on the same browser

## Files

- `index.html` — app shell and fonts
- `styles.css` — responsive neon / glassmorphism visual system
- `app.js` — routes, demo data, reader controls, and publishing interactions

To reset the demo archive, clear the site's local storage in your browser.
